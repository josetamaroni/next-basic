Esto es un proyecto básico de [Next.js](https://nextjs.org/) con los componentes básicos de Shadcn ui.

## ¡Comenzar!

Primero, Ejecutar el servidor de desarrollo:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abrir [http://localhost:3000](http://localhost:3000) en su navegador para ver el resultado.

Puede comenzar a editar la página modificando app/page.tsx. La página se actualiza automáticamente a medida que edita el archivo.

Este proyecto utiliza [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) para optimizar y cargar automáticamente Inter, una fuente personalizada de Google.
